# Pattern-Software-Design

## TokoBeDia
Project ini dibuat untuk Assignment mata kuliah Pattern Software Design 
Website untuk e-commerce dengan fitur sebagai berikut:
1.Guest
  ..- Register
  ..- Login (Data - data user disimpan menggunakan session)
  ..- View Product
2.Member
   ..- View Product
   ..- Profile
   ..- Change Password
3.Administrator
   ..- View User
   ..- View Product
   ..- Insert Product
   ..- Update Product
   ..- View Product Type
   ..- Insert Product Type
   ..- Update Product Type
   ..- Profile
   ..- Update Profile
   ..- Change Password
